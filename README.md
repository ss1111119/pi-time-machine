# Pi Time Machine（π 的時光機）

Pi Time Machine 是一個為高中生與數學愛好者設計的互動式網頁專案，
透過視覺化與實作方式，帶領使用者回顧超過四千年的數學歷史，
理解人類如何一步步逼近圓周率 pi 的精確值。

## Project Features

本專案將圓周率的計算歷史分為四個重要階段，
每一階段皆包含互動模擬與數學概念說明。

### 1. Measurement Era
模擬古巴比倫與古埃及以繩子測量圓周的方式，
理解圓周率的基本定義：圓周長 ÷ 直徑。

### 2. Polygon Approximation
重現阿基米德與祖沖之的割圓術，
透過調整多邊形邊數，觀察周長逐漸逼近圓周。

### 3. Infinite Series
展示 17 世紀的級數方法，
以動態方式呈現萊布尼茲級數如何震盪並逐步收斂。

### 4. Monte Carlo Simulation
使用隨機投點的方式，
以幾何機率與面積比例估算圓周率。

## Mathematical Concepts

本專案涵蓋高中至大學先修程度的數學概念，適合教學展示：

- Geometry  
  圓與內接多邊形周長的關係

- Trigonometry  
  使用正弦函數計算多邊形周長

- Calculus  
  透過反正切函數的級數展開理解 pi 的來源

- Probability and Statistics  
  以圓面積與正方形面積比例估算 pi
